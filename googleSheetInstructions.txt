INSTRUCTIONS FOR GETTING GOOGLE SHEETS INTO FORMAT 

1. EXPORT GOOGLE DOC AS A CSV FILE BY FILE -> DOWNLOAD -> CommaSeperatedValues
2. GO TO THIS URL - https://csvjson.com/csv2json
3. UPLOAD THE CSV FILE YOU JUST ADDED AND CLICK CONVERT.
4. DOWNLOAD THAT FILE AND ADD TO original-files folder 
5. UPDATE THE FILE IN scripts/formatAddress and follow instructions in file
6. AFTER THE NEW FILE IS CREATED IN FILES use jsonWL1.json as an example
7. GO TO scripts/user.js AND FOLLOW INSTRUCTIONS AT THE TOP TO REQUIRE THE NEW FILE CREATED
8. ONCE THAT IS DONE GO TO THAT DIRECTORY AND RUN node user.js to insert into DB
